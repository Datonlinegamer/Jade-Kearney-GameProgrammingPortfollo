    Code Breakdown

    This section of code explains how the character jump, ground detection, and slope movement work:

    • Jump()
      • Makes the player jump if grounded and the jump input is triggered.
      • Ensures a cooldown (jumptracker) is respected to prevent repeated jumping.
      • Resets vertical velocity before applying an impulse force upward.
      • Plays a jump sound effect on successful jump.
    • groundcheck()
      • Uses a BoxCast to check if the player is on the ground.
      • If grounded, sets isGrounded = true and enables gravity.
      • If not grounded, sets isGrounded = false but keeps gravity on.
      • Resets vertical velocity when first touching the ground (to prevent bouncing).
    • DistanceToGround()
      • Performs a Raycast downward to measure how far the player is from the ground.
      • Returns the distance if ground is hit, otherwise returns 0.
    • Slope Handling
      • When airborne and falling, checks the player's distance to the ground.
      • If the player is hovering slightly above a slope, it moves the character downward to maintain ground contact.
      • Prevents floating or bouncing on sloped surfaces by manually adjusting the player's Y-position.
      • Applies friction when grounded to reduce sliding down slopes.

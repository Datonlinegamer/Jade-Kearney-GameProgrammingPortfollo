Code Breakdown

This section of code explains how the character interaction system works:

  • UInteractionComponent handles line tracing and sets CurrentFocusedActor each frame.
  • Actors receive OnBeginFocus and OnEndFocus events.
  • When the player presses the interact key, UInteractionManager:
    • Checks if the actor is interactable.
    • Calls OnInteract() on the object.

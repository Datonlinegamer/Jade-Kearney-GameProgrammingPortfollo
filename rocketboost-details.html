    • Summary

    Rocket Boost is a physics-driven game where the player pilots a retro-futuristic rocket through obstacle-filled environments using precise booster control and momentum. The objective is to collect fuel consumables and special items scattered across each level while avoiding hazards, managing limited fuel, and mastering aerial navigation.

    Core Systems

  • Custom movement using boost thrust, including:
  • Variable acceleration/thrust control
  • Aerial steering and directional control
  • Gravity, drag forces, and ground detection
  • Hovering or mid-air control
  • Boost cooldown or energy management

    • Code Breakdown

    This class handles sound effect playback in Unreal using a singleton pattern. Here's how it works:

  • Singleton Pattern: Ensures only one instance of UUSoundEffectManager exists at runtime via Get().
  • Initialization: Loads specific sound cues using ConstructorHelpers::FObjectFinder in the constructor.
  • Sound Storage: Stores loaded sounds in a SoundEffects map using their names as keys.
  • Playback Function: PlaySoundEffect() plays a sound at a given world location if it exists in the map.
  • Memory Management: Uses AddToRoot() to prevent garbage collection on the singleton instance.

    • Header File (.h):
    Declares the UUSoundEffectManager class, its methods, variables, and static instance. This file defines the class interface that other parts of the game can include and interact with.

    Implementation File (.cpp):
    Defines how the declared functions in the header actually work. It initializes the singleton instance, loads sound cues, stores them in a map, and provides functionality to play them in the game world using UGameplayStatics.

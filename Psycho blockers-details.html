    Code Breakdown

    This section covers wall running and stuck recovery logic for the player character:

    • Wall Running
      • Wall running is activated when the player is near a wall using raycasts.
      • Each frame, Updatewallrun() checks all ground raycast points:
      • 
void Updatewallrun()
{
    wallrun = false;
    foreach (Transform point in groundraycastpoints)
    {
        RaycastHit hit;
        if (Physics.Raycast(point.position, -Vector3.forward.normalized, out hit, 3.6f))
        {
            if (hit.transform.gameObject.GetComponent<Wallrun>())
            {
                wallrun = true;
                rb.useGravity = false;
            }
        }
    }
}
      • If wallrun is active and the player is in the air, gravity is reduced and vertical velocity is slowed:
      • 
if (wallrun && !grounded)
{
    if (rb.velocity.y < 0)
    {
        float brake = .8f;
        if (dagger.swinging) brake = .7f;
        Vector3 v = rb.velocity;
        v.y *= brake;
        rb.velocity = v;
        smoke.Emit(1);
        SA.volume = 10;
    }
}
    • Stuck Recovery
      • If the player is stuck to a surface (isstuck() returns true), a force is applied to push them left and unstick them.
      • 
if (isstuck())
{
    rb.AddForce(Vector3.left * Fowardacceleration * 200, ForceMode.Impulse);
}
